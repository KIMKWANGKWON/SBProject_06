경준 :
예약폼과 예약수정폼에 시각(time) UI 표헌해주기.

단비 :
즐겨찾기 리스트에서 즐겨찾기한 것 delete하기.
이미 회원이 좋아요한 식당은 좋아요가 돼있는 모습의 버튼으로 보이게 하기.
또, 좋아요 버튼을 클릭할 때마다 그 모습이 바뀌도록 하기.



공동 :
root 페이지에 있는 "최근에 등록된 식당", "현재 인기있는 식당" 구현하기.

    최근에 등록된 식당 : 루트컨트롤러에서 model에 레스토랑 테이블을 regdate로 내림차순 정렬한 상위 행 N개를 넣어주면,
    root.jsp에서 그 값들을 받아와, 이미지 소스로 쓰고, 이미지태그에 onclick 속성을 넣어, 상세보기란으로 넘어가게.

    현재 인기있는 식당 : 레스토랑 테이블에 hitcount(readcount) 속성을 추가로 넣어서, 상세보기를 할 때마다 hitcount값을 1 증가시키게 해야함.
    최근등록식당과 비슷한 방식으로 hitcount를 내림차순으로 정렬하여 상위 행 N개를 넘겨주면 됨.
  }
